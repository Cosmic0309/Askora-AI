# Askora - AI Data Automation Platform

A premium AI SaaS landing page built for the Frontend Battle Round 1 Hackathon.

## Features

- Responsive Landing Page
- Bento Grid Features
- Dynamic Pricing Matrix
- Monthly / Annual Billing
- INR / USD / EUR Currency Switcher
- AI Chat Widget
- FAQ Accordion
- SEO Optimized
- Semantic HTML
- Smooth CSS Animations

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
